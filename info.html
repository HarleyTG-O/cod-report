About This Project

This project is dedicated to creating a reporting system for known cheaters in Call of Duty games. By reporting and viewing known cheaters, players can have a better gaming experience.

Project Versions

Main

The main version is the primary and stable version of the project. It includes all the latest features and updates.

Beta

The beta version is for testing new features before they are included in the main version. It may contain bugs and is not recommended for production use.

Go to Beta

Basic

The basic version is a simplified version of the project with essential features only. It is designed for users who need a lightweight version of the project.
